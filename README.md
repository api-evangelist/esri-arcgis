# Esri ArcGIS (esri-arcgis)

Esri ArcGIS is the world's leading geospatial platform providing mapping, spatial analysis, and location intelligence APIs and SDKs. The developer platform includes REST APIs, multiple language SDKs, and cloud services for GIS professionals and developers building location-enabled applications across government, utilities, transportation, infrastructure, and environmental industries.

## APIs

### Esri ArcGIS Platform API

Esri ArcGIS provides geospatial data platform APIs for mapping, spatial analysis, geocoding, routing, and feature services. APIs enable access to hosted feature layers, imagery, elevation data, and spatial analytics for infrastructure, asset management, and environmental monitoring.

**Human URL:** [https://developers.arcgis.com/](https://developers.arcgis.com/)

**Base URL:** `https://www.arcgis.com/sharing/rest`

### ArcGIS Maps SDK for JavaScript

The ArcGIS Maps SDK for JavaScript enables web developers to build compelling 2D and 3D mapping applications with interactive visualizations, spatial analysis tools, geocoding, routing, and portal content management for ArcGIS Online and ArcGIS Enterprise.

### ArcGIS Geocoding Service

The ArcGIS Geocoding Service provides address search, reverse geocoding, address suggestions, and batch geocoding capabilities using the World Geocoding Service.

**Base URL:** `https://geocode.arcgis.com/arcgis/rest/services/World/GeocodeServer`
